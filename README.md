# Jewelry Stores Management System

## Overview
The Jewelry Stores Management System is a robust software solution developed using CodeIgniter, designed to streamline the operations of jewelry stores, making it more efficient in managing inventory, sales, and customer records. This project reflects my commitment to using CodeIgniter for building practical and effective web applications.

## Features
- Inventory management for jewelry items.
- Customer management for tracking purchases and preferences.
- Sales and invoicing system.
- Staff management for employee records.
- Reporting and analytics for store administrators.
- User-friendly interface for jewelry store staff.

## Technologies Used
- PHP (CodeIgniter)
- MySQL
- HTML/CSS
- JavaScript

## How to Use
1. Clone the repository to your local machine.
2. Import the database schema from `database.sql` to set up the required tables.
3. Configure the database connection in `application/config/database.php`.
4. Start a PHP server or use a web server like Apache.
5. Access the application through a web browser.
